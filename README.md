# **SongSphere**

**SongSphere** is a music platform currently in progress. This project aims to provide music enthusiasts with a platform to discover, share and listen to music.

## Technologies used

- BackEnd:
    - Nest.js
    - TypeScript
    - JWT
    - Swagger
- FrontEnd:
    - Next.js
    - React.js
